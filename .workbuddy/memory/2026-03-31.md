## 速率限制措施总结

| 措施 | 实现 | 效果 |
|------|------|------|
| **Session 复用** | requests.Session() 全局单例 | 降低连接开销，提升稳定性 |
| **请求重试** | @_retry_on_failure 装饰器，3 次 1s/2s/4s 退避 | 网络波动时自动恢复 |
| **随机延迟** | 0.3-0.5s 随机间隔替代固定 0.2s | 避免被检测为爬虫 |
| **本地缓存** | JSON 文件，24h 有效期 | API 故障时自动降级，支持离线 |
| **改进日志** | [INFO]/[WARN]/[ERROR] 级别 | 清晰了解执行流程和故障原因 |
